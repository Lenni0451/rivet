# Rivet
Work in progress UI library written in Java.\
The library is in early stages of development, so expect breaking changes and missing features.\
I do not accept big code contributions at this time (smaller changes and bug fixes are welcome).\
However, feel free to experiment with the library and give me feedback and suggestions for features and improvements.

## AI disclosure
I am using AI (Google Gemini and GitHub Copilot) to help me write code for this project.\
I am making sure to review, test and understand/rewrite any code that is generated by AI before using it in the project
to ensure that it meets the quality standards of the project.\
(The only exception at this point is the GridLayout, I will write my own implementation of it once the rest of the library is in a more stable state).
